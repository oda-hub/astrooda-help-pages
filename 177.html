Overview

\n

This page provides a short tutorial for the online analysis interface of SPI-CAS data.

\n

Analysis Flow

\n

The analysis steps are described below:

\n

Query Data

\n
  1. Start analysis by entering the source name of interest in the field "Object name" of the top panel or the times in the cells below.
    2. \n
  2. If you insert a name, click on the "Resolve" button to trigger the resolver. If the object of interest is found, time range fields are filled. You may also wish to fill them manually. The RA and Dec fields are dummy.
    3. \n
  3. Adapt / fill time interval for the data (or enter them).
    4. \n

Timing GRB Analysis

\n
  1. \n

    Select "SPI-ACS" on the instrument quera panel.

    \n
    2. \n
  2. \n

    Check the "Lightcurve" checkbox..

    \n
    3. \n
  3. \n

    Set the time bin width and unit for the lightcurve (minimi is 50 ms). Remind that the start and end time of the lightcurve are determined by the time limits set in the top multi-instrument panel.

    \n
    4. \n
  4. \n

    The "Query Type" field must be set to Real.

    \n
    5. \n
  5. \n

    Launch the lightcurve extraction by pressing the "Submit" button.

    \n
    6. \n

\n

A window pops up showing the status of the processing request. Note that according to the request, the processing time can be quite long. Click on "close" when done to close the window.

\n

\n

Once the request is processed, it is displayed in the workspace, together with other previous results.

\n

\n

Click on the "View" button on the right of the source object to display the lightcurve.

\n

\n

By clicking on the "Download" button, you have a possibility to download the lightcurve in FITS and ROOT format for further analysis.

\n

\n
